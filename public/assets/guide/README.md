# 안내 캐러셀 일러스트

Module B(포스터 제작) 체험 방법 안내 팝업의 4단계 일러스트입니다.
아래 파일명으로 이 폴더에 넣으면 캐러셀에 자동 표시되고, 없으면 이모지로 폴백됩니다.

| 파일 | 단계 | 내용 |
|------|------|------|
| `step1.png` | Step 1 | 주인공이 될 인물의 사진을 찍어주세요 |
| `step2.png` | Step 2 | 만들고 싶은 포스터의 스타일을 선택해주세요 |
| `step3.png` | Step 3 | 세상에 하나뿐인 나만의 영화 포스터가 완성됩니다 |
| `step4.png` | Step 4 | 영화포스터를 저장하거나 전시해보세요 |

- 권장: 정사각형 또는 가로형, 투명 배경 PNG, 높이 240px 이상.
- 경로/개수 변경은 `src/pages/PosterCameraPage.ts` 의 `GUIDE_SLIDES` 에서.
